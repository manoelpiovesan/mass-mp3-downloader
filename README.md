# Usage
## Install Dependencies
### Ffmpeg
```bash
winget install Gyan.FFmpeg
```

### yt-dlp
```bash
winget install yt-dlp.yt-dlp
```

## Links
Put the links into links.txt breaking line

## Start
Click start.bat to start downloading!
